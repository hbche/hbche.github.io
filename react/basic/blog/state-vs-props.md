# React 中 state 和 props 

## 定义

### State
State 是一个可更新的结构，用于包含有关组件的数据或信息，并且可以随时间变化。状态变化可以作为对用户操作或系统事件的响应而发生。它是 react 组件的核心，它决定了组件的行为以及它将如何呈现。状态必须尽可能简单。它表示组件的本地状态或信息。它只能在组件内部或由组件直接访问或修改。

### Props
Props 是只读的，它是一个存储标签属性值的对象，其工作方式类似于 HTML 属性。它允许将数据从一个组件传递到其他组件。Props 类似于函数参数，可以像在函数中传递参数一样传递给组件。Props是不可变的，所以我们不能从组件内部修改 Props。
